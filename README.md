# Twilio オンラインコンテスト 2021

Twilio オンラインコンテスト 2021 に関する紹介スクリプトです。

## インストール&実行

```sh
git clone https://github.com/mobilebiz/twilio-online-contest-2021.git
cd twilio-online-contest-2021
npm start
```
